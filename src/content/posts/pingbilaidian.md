气晕了，老是接到骚扰电话，自从 24 年高考报名之后，就一直有 xx 本科院校，xx 辅导机构的给我打电话，拆哪不是"严厉打击反诈的国家"，"全民实名反诈"，的地方吗。

直到现在（2026 年）都还有 xx 机构，xx 院校的打电话过来，要么就是接通之后没声音直接被挂断的，气死了喵。今天我就教你们一招，让你的手机好吗不可能有电话打来，很简单

这个是让手机号码的所有呼叫都转移的号码，直接在拨号键输入就好了

```
**21*6666666#
```

**温馨提示：** 拨打了之后你就接不到所有电话了，包括你通讯录里的号码，以及家里人的短号等等，反正就是打不进来，就算拨打也会提示`该号码错误`。
所以适合那种没人给你打电话的手机（），或是提前通知亲朋好友，以后咱都用微信等其他方式联系

当然了还有解药,也是在拨号键输入

```
##21#
```

### 原理说明

这是 GSM 标准 USSD 控制码，几乎所有运营商都支持。

```
*21*6666666#
│ │  └── 转移到的目标号码
│ └──── 功能代码：无条件呼叫转移
└────── 指令开始

*21* = 无条件呼叫转移（Call Forwarding Unconditional）
```

为什么你不会接到电话：

```
正常情况：
别人 → 拨你号码 → 运营商 → 你的手机响

呼叫转移之后：
别人 → 拨你号码 → 运营商 → 直接转到 6666666
                               ↑
                         你的手机完全不会参与
```
