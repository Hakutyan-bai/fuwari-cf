## 前言

**Cherry Studio** 是一款轻量且可扩展的开发工具，支持多种插件与集成，非常适合个人开发者与团队协作。

本文将手把手教你如何将 GitHub Copilot 接入 Cherry Studio。

## 环境准备

在开始之前，请确保已完成以下准备：

- **GitHub 账号**（已开通 Copilot 服务）
- 已安装 **Cherry Studio**



## 步骤一：安装 Cherry Studio

https://www.cherry-ai.com/download


## 步骤二：进行 GitHub 授权

1. 安装完成后，点击右上角 **设置按钮**
2. 找到左边的模型服务
3. 找到GitHub Copilot
4. 根据提示授权GitHub
5. 添加模型

## 开始使用吧！
